# Counter App

## Project Overview

This project is a simple Counter Application developed using HTML, CSS, and JavaScript. The application allows users to increase, decrease, and reset a counter value. The purpose of this project is to practice basic JavaScript concepts such as variables, functions, DOM manipulation, events, and conditional statements.

## Features

- Increase counter value
- Decrease counter value
- Reset counter to zero
- Dynamic color changes based on counter value
- Status messages for positive, negative, and zero values
- Responsive and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Folder Structure

Counter-App

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `index.html`.
4. The application will open in your browser.

## JavaScript Concepts Implemented

### Variables
A variable named `count` is used to store the current counter value.

### Functions
A function named `updateCounter()` is used to update the counter display and messages.

### DOM Manipulation
JavaScript accesses and updates HTML elements using `getElementById()` and `innerText`.

### Event Handling
Event listeners are used to detect button clicks.

### Conditional Statements
The application checks whether the counter value is positive, negative, or zero and updates the display accordingly.
